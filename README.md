## 🧠 Project Overview

This application allows users to:

- Browse all **114 Surahs**
- Read **Ayahs with Arabic text and English translation**
- Search verses by **translation text**
- Customize reading experience with **font & size settings**

The goal was to build a **fast, clean, and scalable Quran platform** following modern web development practices.

---

## 🛠 Tech Stack

### Frontend

- Next.js (SSG for performance)
- Tailwind CSS
- LocalStorage (for persistent settings)

### Backend

- Node.js
- Hono (lightweight and fast API framework)

### Database

- Quran dataset collected from public sources

---

## ✨ Features

### 📜 Surah List

- Displays all 114 Surahs
- Arabic & English names
- Clean and responsive UI

### 📖 Ayah Page

- Full Surah with all ayahs
- Arabic text + translation
- Optimized readability

### 🔍 Search Functionality

- Search ayahs using translation keywords
- Fast and user-friendly results

### ⚙️ Settings Panel

- Arabic font selection (multiple fonts)
- Arabic font size control
- Translation font size control
- Persistent settings using localStorage

---

## ⚡ Performance Optimization

- Static Site Generation (SSG) for faster load times
- Optimized API responses
- Minimal and clean UI for better UX
